# 정다희 포트폴리오 (Next.js + Vercel)

첨부 이미지 스타일(레트로 데스크톱 UI) 기반의 포트폴리오 사이트입니다.

## 실행 방법

```bash
cd snu-portfolio
npm run dev
```

브라우저에서 `http://localhost:3000` 를 열어 확인합니다.

## 빌드

```bash
cd snu-portfolio
npm run build
```

## Vercel 배포

이 저장소 루트에 `docs/`가 있고, Next.js 앱은 `snu-portfolio/` 폴더에 있습니다.

- Vercel 프로젝트 생성 시 **Root Directory** 를 `snu-portfolio` 로 지정하세요.
